Посмотреть другие загрузки в архиве.

† Есть два ABI: в Windows используется родная (native) (MSVC) ABI используемая в Visual Studio, и GNU ABI использующая GCC toolchain. Какую версию Rust выбрать зависит от того, какие C/C++ библиотеки вы используете: для взаимодействия с библиотеками, собранными в VS мы используем MSVC билд версию Rust. Для взаимодействия с GNU програмным обеспечением мы используем MinGW/MSYS2 toolchain

MSVC билд Rust дополнительно зависит от Microsoft Visual C++ build tools for Visual Studio 2013 или более поздней версии. Самый простой путь получения утилит сборки является установка Microsoft Visual C++ Build Tools 2015 Другой вариант это установить Visual Studio 2015 или Visual Studio 2013, и во время установки выбрать "C++ tools". Для базового использования сборки GNU не требуется никакого дополнительного ПО.

Поддержка в Rust GNU ABI является более стабильной и рекомендуется для повседневного использования.
